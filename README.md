Munin Alert-Emergency App (UI/UX)
Overview:

Munin Alert is a mobile emergency app concept designed for a company.
I was responsible for the entire process from scratch — starting with
branding and logo design, moving into UI design (colors, typography, and components)
, and finally building complete UX flows and an interactive prototype in Figma.

My Role:

– Branding & Logo Design
– UI Design (Colors, Typography, Buttons)
– UX Flow & Prototyping

Color Palette:.

Color	HEX	Usage
Black	#000000	Main text
Grey	#B2B2B2	Secondary text / backgrounds
Light Grey	#EAEAEA	Neutral backgrounds
Red	#DB3030	Emergency button (Activate Alarm)
Dark Green	#388645	Success / Confirmation
